# 💳 Result University Bank

Консольное приложение на JavaScript, моделирующее банковскую систему с базовой бизнес-логикой.

## Функционал

- ➕ Добавление клиента
- ❌ Удаление клиента (только при нулевом балансе)
- 💰 Пополнение баланса
- 💸 Снятие средств (без ухода в минус)
- 🔄 Переводы между клиентами
- 📊 Получение информации о клиенте

## Особенности реализации

- Использовано **замыкание (closure)** для инкапсуляции данных
- Приватное состояние клиентов (недоступно извне)
- Все операции выполняются только через API банка
- Валидация входных данных (id, amount)
- Использование методов массивов: `find`, `findIndex`, `map`

## Технологии

- JavaScript (ES6+)
