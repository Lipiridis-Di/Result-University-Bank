# Result University Bank

Простой банковский модуль на JavaScript, реализованный через замыкание.

## Функционал

- добавление клиента
- удаление клиента
- пополнение баланса
- снятие средств
- перевод средств между клиентами
- получение информации о клиенте

## Технологии

- JavaScript
- closures
- методы массивов: find, findIndex
- базовая бизнес-логика

## Запуск

Открой файл `index.html` в браузере и посмотри результат в консоли.
